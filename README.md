# World Radio Translator

Live radio streaming with real-time AI translation powered by Gemini.

View your app in AI Studio: https://ai.studio/apps/419d4637-4b7e-4d3e-bde3-840894782169

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env` to your Gemini API key
3. Run the app:
   `npm run dev`
